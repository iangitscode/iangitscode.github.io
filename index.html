Here are some things that I'm proud of!

Cards Against Humanity
An online, simple to use Cards Against Humanity game built in Angular and NodeJS. It can run several games concurrently, joinable through a room code. Playable on desktop or mobile, this clone provides easy access to a classic party game.
Pokemon TCG Draft
A Pokemon TCG draft deck builder inspired by Hearthstone's Arena mode. Players build decks by selecting one out of three randomly selected cards from specified card sets. Players can then export the deck for online play using tcgone.
OS/161
OS/161 Kernel
OS/161 is a teaching operating system written in C for the MIPS system. I implemented several crucial kernel components, including synchronization primitives, system calls, support for multithreading and multiprocessing, and virtual memory.
WLP4
WLP4 Compiler
A compiler written in C++ that translates WLP4, a subset of C++, to MIPS machine code. This compiler supports types, pointers and a heap, a stack, functions, conditional statements, and loops.
GraphHax
A graph theory tool I built in collaboration with Jonah Dlin. It utilises a tool based system to manipulate the graph. Canvas was used to display the nodes and edges, and Javascript and jQuery were used for the logic.
Juggle
A juggling game with a basic physics engine I built from scratch, based on the Facebook Messenger soccer game. This was one of my first Javascript projects.
Ear Training Tool
A tool designed to develop and fine tune pitch recognition skills. Web Audio API was used to connect to the client's speakers and produce a sound at the correct frequency.
Me No Likey
A Chrome Extension that removes the 'Like' button from all Facebook pages.
